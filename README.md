نظام متكامل فائق القوة لأتمتة فحص الثغرات، الاستطلاع الذكي، وإدارة التهديدات السيبرانية.

## 🚀 الميزات الجديدة في النسخة الرابعة (v4):
*   **Live Report Viewer:** عرض نتائج الفحص مباشرة على الشاشة بتنسيق احترافي.
*   **Custom Save Options:** إمكانية اختيار اسم التقرير ومسار الحفظ يدوياً.
*   **Enhanced Fuzzer:** قاعدة بيانات موسعة للمسارات الحساسة (أكثر من مليون ضعف القوة السابقة).
*   **Interactive Interface:** واجهة تفاعلية متطورة مع قوائم تحكم سهلة.
*   **Robust Error Handling:** معالجة متقدمة لكافة أخطاء الاتصال والشهادات.

## 1. محرك الاستطلاع والاستخبارات (Recon & OSINT Module)
هذا المحرك هو "العين" التي ترى بها الأداة كامل نطاق الهدف.

*   **Subdomain Enumeration:**
    *   *التفاصيل:* دمج بين الفحص النشط (Brute-forcing) والسلبي عبر (API) لمواقع مثل `crt.sh`.
    *   *المميزة:* اكتشاف النطاقات الفرعية المخفية التي قد تحتوي على بيئات تطوير (Staging) غير محمية.
*   **Asset Discovery & Fingerprinting:**
    *   *التفاصيل:* التعرف على نظام التشغيل (OS)، نوع الخادم (Nginx/Apache)، وإصدارات اللغات (PHP/Python).
    *   *المميزة:* يمنع "الهجمات العشوائية" ويركز فقط على الثغرات المتوافقة مع تقنيات الهدف.
*   **JS Secret Finder:**
    *   *التفاصيل:* زاحف (Crawler) يقوم بتحميل كافة ملفات Javascript وتحليلها باستخدام Regex.
    *   *المميزة:* استخراج مفاتيح API مسربة، روابط Cloud S3، وكلمات سر مكتوبة داخل الكود (Hardcoded).

---

## 2. محرك فحص الثغرات الفني (Vulnerability Scanning Core)
المسؤول عن تنفيذ الهجمات المحاكية واكتشاف نقاط الضعف.

*   **Advanced Web Fuzzer:**
    *   *التفاصيل:* البحث عن المجلدات والملفات الحساسة مثل `.env`, `.git`, `config.php`.
    *   *المميزة:* يستخدم تقنية "الأتمتة الذكية" لتغيير القوائم (Wordlists) بناءً على نوع الخادم المكتشف.
*   **API Security & IDOR Automator:**
    *   *التفاصيل:* فحص ثغرات الصلاحيات في الـ API عن طريق تبديل المعرفات (Parameters) ومراقبة الاستجابة.
    *   *المميزة:* اكتشاف الثغرات المنطقية التي لا تستطيع الماسحات التقليدية رؤيتها.
*   **Blind Vulnerabilities (Out-of-Band):**
    *   *التفاصيل:* الربط مع خادم خارجي لاكتشاف ثغرات SSRF و Blind SQLi التي لا تعطي رداً مباشراً.
    *   *المميزة:* القدرة على رصد "التفاعل العكسي" من خادم الهدف.

## 3. طبقة التخفي والذكاء الاصطناعي (AI & Evasion Layer)
لضمان استمرارية الفحص دون حظر وبأقصى دقة.

*   **Smart Evasion System:**
    *   *التفاصيل:* تغيير الـ User-Agent بشكل عشوائي واستخدام Proxy Rotation.
    *   *المميزة:* تجاوز أنظمة الـ WAF وجدران الحماية التي تحظر البوتات عند استشعار عدد طلبات كبير.
*   **AI-Risk Scoring:**
    *   *التفاصيل:* استخدام خوارزمية تحليل لترتيب الثغرات المكتشفة من الأشد خطورة إلى الأقل.
    *   *المميزة:* توفير وقت المختبر بالتركيز على الثغرات التي تؤدي لاختراق كامل للنظام أولاً.
*   **Screenshotter & Evidence:**
    *   *التفاصيل:* استخدام Playwright لأخذ لقطات شاشة آلية كدليل على وجود الثغرة.
    *   *المميزة:* تقليل الـ False Positives (النتائج الخاطئة) وتقديم دليل مرئي في التقرير.

---

## 4. نظام التشغيل والإدارة (Orchestration & Reporting)
إدارة العمليات وتوليد المخرجات النهائية.

*   **Asynchronous Task Queue (Celery/Redis):**
    *   *التفاصيل:* توزيع مهام الفحص على عدة "عمال" (Workers) لضمان السرعة الفائقة.
    *   *المميزة:* إمكانية فحص مئات المواقع في وقت واحد دون توقف النظام.
*   **Real-time Notifications:**
    *   *التفاصيل:* تكامل كامل مع Telegram و Discord عبر Webhooks.
    *   *المميزة:* الحصول على تنبيه فوري عند اكتشاف ثغرة "Critical" دون الحاجة لفتح لوحة التحكم.
*   **Automated Professional Reporting:**
    *   *التفاصيل:* توليد تقارير بصيغة PDF و JSON بضغطة زر واحدة.
    *   *المميزة:* تقارير منسقة تشمل (وصف الثغرة، الدليل، خطوات الحل) جاهزة لتقديمها للعميل.
